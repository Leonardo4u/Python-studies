## Descrição

Este repositório é usado para estudar Python. Contém exercícios, scripts e experimentos pessoais que eu criei enquanto aprendo e pratico conceitos da linguagem.

> Objetivo: manter um histórico dos meus exercícios, treinar sintaxe e pequenos projetos.

- `Calculate_diameter.py`

> Nota: os nomes refletem experimentos e exercícios pessoais — podem conter acentuação e formatos variados.

## Como usar

1. Instale o Python 3.8+ (recomendado).
2. Navegue até a pasta do repositório no terminal.
3. Execute qualquer script com:

```bash
python nome_do_arquivo.py
```

Exemplo:

```bash
python cadastroUser.py
```

## Organização e boas práticas (sugestões)

- Use virtualenv/venv para isolar dependências.
- Adote nomes de arquivos sem espaços e sem caracteres especiais para scripts que você pretende publicar.
- Crie um `requirements.txt` quando adicionar bibliotecas externas.
